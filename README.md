# NotesprojectApp

A simple, responsive Notes Application where users can create, edit, delete, and organize notes into categories. Built using HTML, CSS, and JavaScript.

Features:
Create, edit, and delete notes
Category-wise note organization
Search notes
Light & Dark mode
Clean and responsive UI

 Project Structure:
/project-folder
│── index.html
│── style.css
│── script.js
│── assets/
